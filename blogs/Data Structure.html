Dyanmic Arrays

Dynamic arrays, also known as vectors in C++ or ArrayLists in Java, are versatile data structures capable of holding a variable number of elements of the same data type. Unlike static arrays, dynamic arrays can expand in size by doubling their capacity when necessary. New elements can be swiftly added to the end of the dynamic array until the available space is exhausted. When additional room is needed, the underlying fixed-sized array is enlarged, which can be a costly operation due to the creation of a larger array and copying of existing elements. Other terms for dynamic arrays include array list, growable array, resizable array, and mutable array. In summary, dynamic arrays overcome the fixed size limitation of regular arrays by automatically resizing as more elements are added, eliminating the need to specify the size in advance. 

            
#include <iostream>
#include <vector>

int main() {
    // Create a dynamic array using std::vector 
    std::vector dynamicArray;

    // Add elements to the dynamic array
    dynamicArray.push_back(1);
    dynamicArray.push_back(2);
    dynamicArray.push_back(3);

    // Access elements
    std::cout << "Element at index 0: " << dynamicArray[0] << std::endl;

    return 0;
}
